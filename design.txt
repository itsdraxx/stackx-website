2. TYPOGRAPHY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Font Family               Token               Weights Loaded
  ─────────────────────────────────────────────────────────────
  Inter (Google Fonts)      --font-body          Variable (400–700)
  Poppins (Google Fonts)    --font-heading       400, 500, 600, 700, 800

Loading Strategy:
  • Both fonts use `display: "swap"` for fast rendering.
  • CSS variables (--font-inter, --font-poppins) are set via Next.js font API.
  • Body defaults to Inter via font-family on <body>.
  • Headings use Poppins via inline style overrides on <h1>–<h4> elements.

Typography Scale (Tailwind):
  • Hero headline     : text-4xl → sm:text-5xl → md:text-6xl → lg:text-7xl
  • Section titles    : text-3xl → sm:text-4xl → lg:text-5xl
  • Card headings     : text-lg → text-xl
  • Body text         : text-base → sm:text-lg
  • Small/labels      : text-sm, text-xs
  • Spacing           : leading-tight (headings), leading-relaxed (body)
  • Tracking          : tracking-tight (hero), tracking-wider/widest (badges)

5. CUSTOM CURSOR
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

File: src/components/ui/CustomCursor.tsx

  • Two-element cursor: a small dot + a larger ring that follows with spring delay
  • Dot: 8×8px, bg-primary (#8B5CF6), rounded-full, mix-blend-difference
  • Ring: 40×40px, border-primary/50, follows with spring (damping:25, stiffness:200)
  • On hover over interactive elements (a, button, input, etc.):
    – Ring expands to scale(1.5) and opacity reduces
    – Dot shrinks slightly
  • On click: scales down for a "press" effect
  • Disabled on touch devices (detects ontouchstart / maxTouchPoints)
  • Hidden when cursor leaves the viewport
  • The system cursor is hidden globally (`cursor: none` on body)

6.1 Framer Motion (component-level)
────────────────────────────────────
  • Fade-up entrance: initial={{ opacity:0, y:30 }}, whileInView={{ opacity:1, y:0 }}
    – Used on: SectionHeading, GlassCard, hero elements, grid items
    – viewport={{ once: true, margin: "-50px" }}
    – duration: 0.4–0.6s, staggered with delay (i * 0.08)
  
  • Scale entrance: initial={{ opacity:0, scale:0.8 }}, whileInView={{ opacity:1, scale:1 }}
    – Used on: AnimatedCounter, average rating section

  • Slide-in: initial={{ opacity:0, x:±30 }}, whileInView={{ opacity:1, x:0 }}
    – Used on: About page two-column layout (story left, stats right)

  • Mobile menu: AnimatePresence with height:0→auto, opacity:0→1

  • Testimonial carousel: AnimatePresence with opacity/x transitions,
    auto-advances every 5 seconds

6.2 CSS Keyframe Animations (globals.css)
─────────────────────────────────────────
  • float         : translateY(0) → translateY(-20px) rotate(5deg), 6s ease-in-out infinite
  • floatReverse  : translateY(0) → translateY(20px) rotate(-5deg), 7s ease-in-out infinite
    – Applied to decorative floating shapes on the hero section
  
  • marquee       : translateX(0) → translateX(-50%), 30s linear infinite
    – Applied to the client logos ticker (infinite horizontal scroll)
  
  • pulseGlow     : box-shadow oscillates between 20px and 40px spread, 3s infinite
    – Applied to the primary CTA button ("Get Free Consultation")

6.3 Transition utilities
────────────────────────
  • transition-all duration-300 : default for interactive elements
  • transition-all duration-500 : navbar background on scroll
  • transition-colors           : nav links, footer links
  • transition-transform        : value icons on hover (group-hover:scale-110)
  • hover:-translate-y-0.5      : subtle lift on CTA buttons

10. GLOBAL DESIGN PATTERNS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

10.1 Glassmorphism
──────────────────
  The defining visual language of the site. Every card uses the .glass-card class:
    • Semi-transparent dark background
    • Heavy backdrop blur (20px)
    • Subtle purple-tinted border
    • Hover: border brightens, soft glow shadow

10.2 Gradient Accents
─────────────────────
  • Purple→Deep purple: CTAs, primary buttons, active states
  • Cyan→Dark cyan: secondary accents, mission icons
  • Purple-light→Cyan: gradient-text for headlines, counter values
  • All gradients use 135° angle for consistency

10.3 Badge/Pill Pattern
───────────────────────
  Repeated across all pages for section labels, categories, and tags:
    • px-4 py-1.5, text-xs, font-medium, tracking-wider, uppercase
    • rounded-full
    • bg-primary/10, text-primary-light, border border-primary/20

10.4 Hero Section Pattern
─────────────────────────
  Every page follows the same hero structure:
    • pt-24 (nav clearance) + py-16
    • Radial gradient overlay (purple or cyan, 8% opacity)
    • SectionHeading (badge + title + subtitle)
    • relative overflow-hidden for gradient containment

10.5 Form Input Pattern
───────────────────────
  Consistent across contact and careers pages:
    • bg-white/5, border border-white/10, rounded-xl
    • text-sm, text-white, placeholder:text-muted/50
    • Focus: border-primary/50, ring-1 ring-primary/25
    • Error: border-error (red)
    • Labels: text-sm font-medium text-white

10.6 Icon Container Pattern
───────────────────────────
  For feature icons throughout the site:
    • w-10 h-10 (or w-14 h-14 for larger)
    • rounded-lg (or rounded-2xl)
    • bg-gradient-to-br from-primary to-primary-deep
    • Icon inside: text-white
    • Some use hover:scale-110 via group-hover

15. DESIGN PHILOSOPHY & PRINCIPLES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. DARK-FIRST, PREMIUM FEEL
     The entire site uses a single dark theme with deep navy-black backgrounds.
     This creates contrast with the vibrant purple and cyan accents, giving the
     site a modern, premium, tech-forward look.

  2. GLASSMORPHISM AS VISUAL LANGUAGE
     Every card, panel, and container uses the same glassmorphism treatment:
     translucent backgrounds with blur and subtle purple-tinted borders. This
     creates visual depth and layering without harsh edges.

  3. GRADIENT ACCENTS FOR HIERARCHY
     Purple-to-deep-purple gradients mark primary interactive elements.
     Cyan accents provide secondary emphasis. Gradient text highlights key
     phrases in headings, creating visual interest without additional imagery.

  4. MOTION WITH PURPOSE
     Animations serve clear purposes: fade-up entrances establish content
     hierarchy as users scroll, floating shapes add life to the hero, and
     the pulsing glow on the primary CTA draws attention to the key action.

  5. CONSISTENT SPACING & SCALE
     All sections use the same max-w-7xl container, consistent py-16/py-20
     vertical rhythm, and a clear typography scale from 7xl hero headlines
     down to xs labels.

  6. RESPONSIVE BY DEFAULT
     Every layout uses Tailwind's responsive grid system with clear breakpoints.
     The navbar transforms from horizontal to a mobile drawer. Grids collapse
     from 4→2→1 columns. Typography scales down gracefully.

  7. ACCESSIBILITY CONSIDERATIONS
     – All interactive elements have cursor-pointer
     – Focus-visible outlines use brand purple
     – Aria labels on icon buttons and social links
     – Semantic HTML (nav, main, footer, section)
     – Form validation with clear error messages
